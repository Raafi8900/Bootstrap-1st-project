Captain Code Editor

This platform comes with a multi-language supported code editor, which
will make your programming practice even easier.

Captain Code Playground is lightweight and fast enough to work on all modern browsers. It supports different programming languages, thus helps in pointing out code errors, and provides automatic code completion and suggestions.

In Captain Code Playground you can use different programming languages ​​like Python, Java, C++, JavaScript and PHP. It provides syntax highlighting and code completion features specific to programming languages. It is also fully compatible with other programming features that provide convenience for writing code.

The code playground connects the test-engine with the code-testing step to the platform's programming challenges. This means, when you run the code, it will automatically follow the code-testing steps.
